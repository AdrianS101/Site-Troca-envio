# PRD - TROCAENVIO

## Descrição do Projeto
Site institucional moderno, responsivo e de alta conversão para TROCAENVIO - uma solução logística inovadora focada em devolver tempo às pessoas através de lockers inteligentes em condomínios.

## Identidade Visual
- **Cor Principal:** #133660 (Azul escuro - confiança)
- **Cor Destaque:** #00B6A5 (Verde turquesa - inovação/CTAs)
- **Cor Base:** #FFFFFF (Branco - respiro e clareza)
- **Tipografia:** Montserrat (Bold/SemiBold para títulos, Regular para texto)
- **Estilo:** Clean, minimalista, espaçado, moderno

## Posicionamento
A TROCAENVIO não é apenas logística - é uma solução de liberdade e praticidade que:
- Elimina tempo perdido com logística
- Simplifica o envio de encomendas
- Traz conveniência para dentro do condomínio
- Integra com Mercado Livre, Shopee e Correios

**Mensagem Central:** "A TROCAENVIO não entrega pacotes. Ela devolve tempo para você viver o que realmente importa."
